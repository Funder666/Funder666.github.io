博客添加Live2D人物

随笔
发布日期:   2021-09-03
文章字数:   550
阅读时长:   2 分

在苦心折腾了数小时之后，终于成功地在博客上添加上了动漫人物。网上的教程实属纷繁复杂，我在这里进行简单的概括总结。

一.添加官方模型

官方模型共有21种，感觉耐看的只有三个（koharu,haruto,z16)，模型的预览可以去这个网站，传送门。

这里我针对koharu模型进行讲解，也就是我博客主页右下角的小可爱

  1. 进入博客根目录后用git命令安装插件
npm install --save hexo-helper-live2d
  1. 安装动漫人物库
npm install live2d-widget-model-koharu #想添加其他的就改最后的模型名字，名字在模型预览网站可以找到
  1. 在博客根目录下的_config.yml配置文件中添加如下代码
live2d: 
  enable: true 
  scriptFrom: local 
  pluginRootPath: live2dw/ 
  pluginJsPath: lib/ 
  pluginModelPath: assets/ 
  tagMode: false 
  log: false 
  model: 
    use: live2d-widget-model-koharu #填加你想用的模型 
  display: 
    position: right 
    width: 150 
    height: 300 
  mobile: 
    show: true 
    react: 
      opacity: 0.7

最后hexo g -d 后，刷新一下博客就能出来了

二. 添加大佬们自己定制的模型

我这里找到的是添加恶魔小萝莉的模型

2.1 添加固定样式的模型

首先找到head.ejs文件，我的在 E:\blogtest\themes\matery\layout_partial\head.ejs

不同主题位置可能不一样，这里针对matery主题

接着添加如下两行代码:

    <link rel="stylesheet" type="text/css" href="//cdn.jsdelivr.net/npm/font-awesome/css/font-awesome.min.css">
    <script src="//cdn.jsdelivr.net/gh/stevenjoezhang/live2d-widget/autoload.js"></script>

保存配置文件，最后hexo g -d 后，刷新一下博客就能出来了。

2.2 添加定制化的模型

上面的方法添加的模型样式大小位置语言等都是固定的，想要更换模型可以调用其他的API，想要完全自定义模型，需要自行研究后端API，太难了，希望有人能教教。

可参考以下网址，我表示没整明白，但的确有人根据他们的方法整好的。

大佬可以将各种游戏的2d人物提取出来，变成自己博客的看板娘，羡慕~
