概述

​ 当我们点开一个微信小程序的时候，其实是微信已经将它的源文件从服务器上下载到了手机，然后再来运行的。所以我们应该可以从手机本地找到到已经下载过的小程序文件（apk包），但用手机自带的文件管理软件一般是找不到的，所以需用到RE管理器进行寻找，找到后放到本地，再借用大神的反编译脚本（配合npm依赖）进行解析即可

准备工作

1 node.js 运行环境

下载地址

下载地址

3 夜神模拟器

下载地址

4.RE管理器（百度下载安卓版，会自动下载进夜神模拟器中）且在夜神模拟器中安装好QQ微信

具体步骤

  • 登录夜神模拟器，到模拟器内部设置超级用户权限，将自动相应设置为允许（

这些操作的目的都是为了能让RE管理器顺利的获取到ROOT权限）

  • 在模拟器中登录好QQ微信，接着在微信中搜索想抓取的小程序，小程序运行后，apk包即已下载入手机本地

  • 进入RE管理器，进入目标位置：

    根目录/data/data/com.tencent.mm/MicroMsg/{一串16进制字符}/appbrand/pkg/

    会有两个后缀为wxapkg的文件，一般最小的那个就是我们要找的文件（一般小于1M)

    长按该文件，右上角三个点中里选择压缩，再发送至我的电脑（QQ接收）（电脑端的QQ也需开着接收，然后将文件存入电脑端本地），此时回到电脑端本地

  • cmd 进入 反编译脚本的目录，执行npm install

npm install是根据反编译脚本目录中package.json文件指定的依赖进行安装，所以如果你下载的脚本并没有该文件，或者package.json文件内容有误,会导致npm install失败,此时就需手动npm需要的各个依赖,但经过我自己的尝试后发现就算手动安装完,最后反编译时还是会报错,因此我找了很久的能正确npm install的反编译脚本,已附上

  • 依赖安装完后,需修改反编译文件目录下的wuWxss.js文件(这是由于微信升级导致旧版的脚本已经无法翻编译wxss文件了)
